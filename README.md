# 西浦体育中心场地预订助手

一个半自动预订网球场和羽毛球场的 Python 应用程序，提供图形界面和自动化抢先预订功能。

XJTLUer苦于午夜手速慢已久，我们可以利用一些帮助达到抢场地的目的。

## 系统要求

- Python 3.7+
- Microsoft Edge 浏览器
- Edge WebDriver（通常随 Edge 浏览器自动安装）

## 功能特性

- 🎾 支持网球场和羽毛球场预订
- ⏰ 自动定时触发预订
- 🖥️ 友好的图形界面
- 🔄 自动化预订流程

## 技术栈

- Python 3
